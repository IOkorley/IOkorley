## Current Research

RECURIA is a mathematically grounded feature engineering framework that investigates whether instability and transition dynamics can be represented through a common mathematical structure across unrelated domains.

Validation studies include:

- Clinical Deterioration Prediction
- Multiple Sclerosis Conversion Prediction
- Autoimmune Disorder Classification
- Solar Activity Prediction
- Groundwater Stress Forecasting
- Retail Demand Forecasting
- Bike-Sharing Demand Forecasting
